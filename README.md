
## API Reference

#### Get all items

```http
  GET /dashboard
```
#### Search

```http
  GET /dashboard/search 
  GET /dashboard/search/:carSize
```
#### Create
```http
  GET /dashboard/create 
  POST /dashboard/create
```
#### Update
```http
  GET /dashboard/update/:id
  POST /dashboard/update/:id
```
#### Delete
```http
  POST/dashboard/delete/:id
```
